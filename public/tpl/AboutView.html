SISTEMA DE ATENDIMENTO A CLINICA DE FISIOTERAPIA

UNIVERSIDADE SÃO CAETANO DO SUL


Esta Aplicação foi desenvolvida com fins acadêmicos para que os gestores e professores
possam acompanhar os processos realizados em forma de indicadores, além de disponibilizar
para a Clínica uma forma mais segura e ágil de efetuar os cadastros de seus pacientes,
atualizar e acompanhar corretamente os tratamentos realizados e a situação atual dos
mesmos.

Utilizamos para o desenvolvimento o banco de Dados MongoDB orientado à documentos
e um ambiente Node.js (linguagem JavaScript no servidor).
